How to Use PWPush Extension

Quick Share (Home Page)

Follow these steps to quickly share content using the home page:

  1. Enter the text or password you want to share in the text field on the home page.
  2. Click the "Submit to PWPush" button.
  3. A link will be generated. You can copy the link using the "Copy" button next to it.
  4. Send the link to the person you want to share the password with.

Context Menu Share & Settings

The PWPush extension also allows you to securely share text directly from the context menu. Here's how:

  1. Highlight the text or password you want to share on any webpage.
  2. Right-click the highlighted text and select the "PWPush" option from the context menu.
  3. A link will be generated and copied to your clipboard automatically.
  4. Paste the link and share it with the recipient.

For advanced options, you can configure the number of views, expiration days, and other settings on the Settings page:

  • Set how many days until the link expires.
  • Choose the number of views allowed before the link is no longer valid.
  • Enable or disable the one-step link option.
  • Allow the content to be deletable by the viewer.

Security & Encryption

PWPush ensures the security and privacy of your shared content:

  • Content is encrypted prior to storage and is available only to those with the secret link.
  • All content and files of the push are deleted entirely upon expiration.
  • An audit log of activity is kept and available indefinitely for logged-in users.
